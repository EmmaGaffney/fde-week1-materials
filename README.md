# FDE Week 1 Materials
 
This repository contains Week 1 bootcamp reference materials for the FDE AI Bootcamp.
 
Purpose:
- Store the shared Week 1 markdown guidance files
- Provide reference material to a CodeMie assistant
- Support production of the five required deliverables:
  1. Problem statement & success metrics
  2. Delegation analysis
  3. Agent specification
  4. Validation design
  5. Assumptions & unknowns
 
Note:
These materials are reference context only. Scenario-specific outputs should be generated separately based on the sealed scenario provided during the assessment.
